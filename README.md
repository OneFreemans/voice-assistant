# Голосовой ассистент Oleg

Простой голосовой ассистент на Python. Умеет слушать команды, отвечать голосом, работать с ВК, показывать погоду, курс валют, запускать программы и ставить таймеры.

---

## 🚀 Возможности

| Команда | Что делает |
|--------|-----------|
| `Джарвис, сколько время` | Говорит текущее время |
| `Джарвис, какой сегодня день` | Показывает дату |
| `Джарвис, погода` | Погода в Кемерово |
| `Джарвис, курс доллар` | Курс доллара к рублю |
| `Джарвис, курс евро` | Курс евро к рублю |
| `Джарвис, таймер 5 минут` | Ставит таймер |
| `Джарвис, расскажи анекдот` | Случайный анекдот |
| `Джарвис, запусти steam` | Запускает программу |
| `Джарвис, открой вк` | Открывает сайт в браузере |
| `Джарвис, отправь сообщение Имя Фамилия текст` | Отправляет сообщение ВК |
| `Джарвис, последнее сообщение` | Читает последнее сообщение ВК |
| `Джарвис, ответь на сообщение текст` | Отвечает на последнее сообщение ВК |
| `Джарвис, рассчитай стяжку` | Расчёт материалов для стяжки |
| `Джарвис, сердце 5 красное` | Рисует сердечки в консоли |
| `Джарвис стоп` | Завершает работу |

## 🛠️ Установка
1. Клонируй репозиторий:
```bash
git clone https://github.com/OneFreemans/voice-assistant.git
cd voice-assistant
```
---
2. Создай виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate      # Linux/Mac
.venv\Scripts\activate          # Windows
```
---
3.Установи зависимости:
```bash
pip install -r requirements.txt
```
---
4.Настрой переменные окружения:
```bash
Скопируй .env.example в .env
```
---
5.Вставь:
```bash
VK_TOKEN=ваш_токен
MY_USER_ID=ваш_id   
```
---
6.Запусти ассистента:
```bash
python Oleg/main.py
```
