# Голосовой ассистент Oleg

Простой голосовой ассистент на Python. Умеет слушать команды, отвечать голосом, работать с ВК, показывать погоду, курс валют, запускать программы и ставить таймеры.

---

## 🎤 Активация и распознавание

Активация происходит по ключевому слову **"Олег"**. После активации можно давать команды.

### Нечёткое распознавание (Fuzzy Matching)

Для повышения надёжности используется нечёткое сопоставление строк на основе алгоритма Левенштейна (SequenceMatcher). Это позволяет распознавать команду активации даже при:
- Опечатках: "ОлеГ", "ОлЕг", "олег"
- Произношении: "Олеже", "Олежа", "Олежка"
- Похожих звуках: "Олич", "Олиг"

Порог схожести: 78% — баланс между ложными срабатываниями и пропуском правильных команд.

Изменить порог можно в `utils/fuzzy.py`:
```python
FUZZY_THRESHOLD = 78  # 0-100, меньше = мягче
```

## 🚀 Возможности

| Команда                                        | Что делает |
|------------------------------------------------|-----------|
| `Олег, сколько время`                          | Говорит текущее время |
| `Олег, какой сегодня день`                  | Показывает дату |
| `Олег, погода`                              | Погода в Кемерово |
| `Олег, курс доллар`                         | Курс доллара к рублю |
| `Олег, курс евро`                           | Курс евро к рублю |
| `Олег, таймер 5 минут`                      | Ставит таймер |
| `Олег, расскажи анекдот`                    | Случайный анекдот |
| `Олег, запусти steam`                       | Запускает программу |
| `Олег, открой вк`                           | Открывает сайт в браузере |
| `Олег, отправь сообщение Имя Фамилия текст` | Отправляет сообщение ВК |
| `Олег, последнее сообщение`                 | Читает последнее сообщение ВК |
| `Олег, ответь на сообщение текст`           | Отвечает на последнее сообщение ВК |
| `Олег, рассчитай стяжку`                    | Расчёт материалов для стяжки |
| `Олег, сердце 5 красное`                    | Рисует сердечки в консоли |
| `Олег стоп`                                 | Завершает работу |

## 🛠️ Установка
1. Клонируй репозиторий:
```bash
git clone https://github.com/OneFreemans/voice-assistant.git
cd voice-assistant
```
---
2. Создай виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate      # Linux/Mac
.venv\Scripts\activate          # Windows
```
---
3.Установи зависимости:
```bash
pip install -r requirements.txt
```
---
4.Настрой переменные окружения:
```bash
Скопируй .env.example в .env
```
---
5.Вставь:
```bash
VK_TOKEN=ваш_токен
MY_USER_ID=ваш_id   
```
---
6.Запусти ассистента:
```bash
python Oleg/main.py
```
